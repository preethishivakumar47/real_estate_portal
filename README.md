# Real Estate Portal 🏠

**Real Estate Portal** is a full-stack web application that allows users to **explore, list, buy, sell, and rent properties online**. It connects **buyers, sellers, and agents** in a secure, user-friendly platform with advanced search and filtering capabilities. :contentReference[oaicite:2]{index=2}

---

## 🚀 Features

### 🏡 User-Side
- Browse property listings with images and details
- Search properties by filters like location, price range, type
- Register & login to save favorite properties
- Contact property owners or agents

### 👨‍💼 Sellers & Agents
- Add new property listings
- Edit or delete your property posts
- Upload property images

### 🔐 Admin Panel
- Manage property listings
- Approve or remove new posts
- View user activity
